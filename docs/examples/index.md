# Examples

The examples are small integration files, not complete applications.

| Example | Purpose |
| --- | --- |
| `examples/telegram_gateway.py` | Programmatic gateway startup from env settings. |
| `examples/pydantic_acp_agent.py` | Pydantic AI agent target for ACP Kit. |
| `examples/langchain_acp_graph.py` | LangChain graph target for ACP Kit. |
| `examples/acprouter_with_acpkit_instance.py` | In-process ACP Kit adapter object mounted into Telegram. |
| `examples/acpremote_bridge_server.py` | Remote WebSocket bridge for an ACP agent. |
| `examples/acprouter_remote_acp.py` | Local Telegram router connected to a remote ACP endpoint. |

See [Agent Server Setup](../getting-started/agent-server.md) for end-to-end commands.
